# 04. 画面構成と遷移仕様

## 1. 画面一覧
ゲームは以下の独立した画面で構成される。

1. **タイトル**
2. **タイム記録**
3. **ステージ選択**
4. **ゲーム**
5. **タイム、ランキング**

---

## 2. 各画面の機能および遷移ルール

### 2.1. タイトル
- **機能**: ゲームのポータル画面。ゲスト名（未登録の場合は初回起動時に登録ダイアログを自動表示）を管理する。
- **UI・機能**:
  - **プレイヤー名表示**: 現在登録中の名前を表示する。
  - **名前変更ボタン**: 登録したゲスト名をいつでも自由に変更できる入力ダイアログを表示する。
- **配置ボタン**:
  - **ステージ選択へ移動**: 「ステージ選択」画面へ遷移する
  - **タイム記録へ移動**: 「タイム記録」画面へ遷移する

### 2.2. ステージ選択
- **機能**: プレイするステージ（3〜5個）を選択する画面。
- **配置ボタン**:
  - **タイトルに戻る**: 「タイトル」画面へ遷移する
  - **ステージ決定（各ステージボタン）**: 対象のステージで「ゲーム」画面へ遷移する

### 2.3. タイム記録
- **機能**: これまでの個人記録（ベストタイムなど）および他プレイヤーのランキングを確認する画面。
- **配置ボタン**:
  - **タイトルに戻る**: 「タイトル」画面へ遷移する

### 2.4. ゲーム
- **機能**: 実際の3Dアクションレースプレイ画面。
- **ゲーム開始演出**:
  - ステージを選択してゲーム画面に切り替わった後、**5秒間のカウントダウン**（画面に「5」「4」「3」「2」「1」と表示）を行ってからレースを開始する。
  - カウントダウン中は、ゲームタイムの計測やプレイヤー、CPUの物理移動（進行、左右操作、アクション）は停止する。
  - ポーズメニューからの「リスタート」時にも、同様に5秒のカウントダウンを行ってから開始する。
- **ポーズ機能**: プレイ中にゲームを一時停止できるシステム
- **ポーズ中メニュー**:
  - **ゲームに戻る**: ポーズを解除し、ゲームプレイを再開する
  - **ステージ選択に戻る**: プレイを中断し、「ステージ選択」画面へ遷移する
  - **リスタート**: 現在のステージをはじめからやり直す
- **ゲームクリア時**: ステージクリア（ゴール到達）時に自動的に「タイム、ランキング」画面へ遷移する

### 2.5. タイム、ランキング
- **機能**: ステージクリア後に、今回のプレイタイムとランキングを表示する結果画面。
- **配置ボタン**:
  - **タイトルに戻る**: 「タイトル」画面へ遷移する
  - **ステージ選択に戻る**: 「ステージ選択」画面へ遷移する
