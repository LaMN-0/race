# 01. コアメカニクス

## 1. ゲーム概要
パルクールをモチーフとした、3D空間におけるスピード感と躍動感のある「3Dアクション×レース」ゲーム。

## 2. 画面およびカメラ仕様
- **画面の向き**: 横向き（ランドスケープ）固定
- **カメラ視点**: TPS（三人称視点）

## 3. アクションシステム
パルクールらしさを表現するため、以下のコアアクションを搭載する。

### 3.1. ジャンプ
- 地上およびスライディング中に実行可能。段差を飛び越えたり、高い足場へ移動する。

### 3.2. スライディング
- 前進移動の慣性を維持したまま、姿勢を低くして滑り進むアクション。
- 低い障害物の隙間（くぐり抜け穴）を通過するために使用する。

### 3.3. レッジグラブ（崖登り / 縁に掴まる）
- 空中で足場や壁の「端（縁）」に接触した際、自動的に端を両手で掴み、そのまま自動で上まで登りきる（オートアクション）。

---

## 4. 操作方法
プレイヤーの操作環境（PC、スマートフォン）において、制限されたキー/タップおよびジェスチャーのみで完結する設計とする。

### 4.1. PC向け（キーボード＆マウス）
- **移動**: W (前進), A (左), S (後退), D (右) キー
- **視点操作**: マウス
- **ジャンプ**: Space キー
- **スライディング**: Shift キー
- **レッジグラブ**: 壁の縁に接触時に自動発動し、自動で登る。

### 4.2. スマートフォン向け（タッチ操作）
- **移動**: 画面上に配置された「仮想パッド（バーチャルパッド）」
- **ジャンプ**: 画面のタップ
- **スライディング**: 画面の下フリック操作
- **レッジグラブ**: 壁の縁に接触時に自動発動し、自動で登る。
